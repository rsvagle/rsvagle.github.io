Fantasy Reference

This Java project used Swing and a MySQL database I created to give the user a list of the top 60 fantasy football players and included their relevant information and statistics. Users can navigate through the list, which is organized by player ranking. While doing this project I was able to lot about the patterns and interfaces (and why we use them) commonly used with Java Swing and how to connect a Java project to an existing database with the JDBC API.
