Draft Day Divas

Draft Day Divas is a project that hosts a world of fantasy football players that's complete with stats, game logs, teams, etc. The source code consists of an Angular frontend and my first Django backend project. My long term intention is to create a machine learning model to evaluate weekly statistics and make a prediction for a player's fantasy score in the next week.
